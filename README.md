## Development

1. Open this repository in VSCode
2. Press `F5` to launch the Extension Development Host
3. Select **Yozakura** from `Preferences: Color Theme`
4. Edit [`themes/Yozakura-color-theme.json`](./themes/Yozakura-color-theme.json) - changes reload live
